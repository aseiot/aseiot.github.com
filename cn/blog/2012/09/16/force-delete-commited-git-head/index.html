Git删除错误提交

Sep 16th, 2012

在使用git时，免不了偶尔会提交错误，如果是本地，我们可以直接reset即可，但有时已经提交到远端服务器如GitHub，那么怎么去强制重置此类错误提交呢？

方法:

  • 首先本地hard reset:

git命令：

git reset --hard <commit_id>

Git-Extension操作：

右键选择需要重置的提交，然后选择reset current branch to here，弹出窗口中重置类型选择hard reset：

  • 然后强制提交到远端:

git命令：

git push origin HEAD --force

Git-Extension操作：

选择commands菜单下push或使用CTRL+UP快捷键调出push窗口，此时注意点击Show options，然后勾选Force Push，提交即可。

OK! 搞定！
